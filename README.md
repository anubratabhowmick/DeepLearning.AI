# DeepLearning.AI Notebooks

Hands-on exercises and reference notebooks for Andrew Ng's latest DeepLearning.AI offerings. Each course lives in its own folder so you can focus on a single curriculum without juggling conflicting dependencies.

## Repository Layout
- `agentic_ai/` – Agentic AI course materials and supporting files. See the folder README for details.
- `LICENSE` – MIT license for this collection (mirrors upstream course licenses when available).

## Courses
- [Agentic AI](agentic_ai) – covers workflow design for tool-augmented LLM agents.

Each course directory includes its own README with setup steps, notebook guidance, and any course-specific notes.
